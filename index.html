    The webcam captures frames → faces are detected using Haar Cascade → emotions are simulated → study suggestions are generated → productivity is tracked and logged.

    C++ Project Files

    main.cpp

    Entry point of the program. Opens webcam, runs the main loop, handles keyboard input (q to quit, m to toggle mode), and coordinates all other modules.

    face_detection.h / .cpp

    FaceDetector class — loads the Haar Cascade XML file and uses it to find faces in each video frame. Draws green rectangles around detected faces.

    emotion_simulator.h / .cpp

    EmotionSimulator class — uses rand() to pick a random emotion (Happy, Sad, Angry, Neutral, Sleepy) every ~2 seconds to simulate real detection.

    decision_engine.h / .cpp

    DecisionEngine class — takes an emotion and returns a study suggestion. Supports Strict mode (strong warnings) and Relaxed mode (gentle tips).

    productivity_tracker.h / .cpp

    ProductivityTracker class — tracks focused vs distracted count, calculates productivity score, manages the study timer, prints BEEP alerts, and logs emotions to log.txt.
