🎓 Emotion-Based Study Assistant

C++ & OpenCV Project — Runs locally on your laptop webcam

📁 Project Files

  • main.cpp — Main program loop
  • face_detection.h/.cpp — Haar Cascade face detection
  • emotion_simulator.h/.cpp — Random emotion simulation
  • decision_engine.h/.cpp — Study suggestions + modes
  • productivity_tracker.h/.cpp — Score, timer, logging

⚡ How to Run

1. Clone this repo to your laptop

2. Install OpenCV (brew install opencv / sudo apt install libopencv-dev)

3. Copy haarcascade_frontalface_default.xml into the folder

4. Compile & run:

g++ main.cpp face_detection.cpp emotion_simulator.cpp decision_engine.cpp productivity_tracker.cpp -o app `pkg-config --cflags --libs opencv4`
./app

🎮 Controls

  • q — Quit & show session summary
  • m — Toggle Strict / Relaxed mode

✨ Features

  • Webcam face detection with rectangle overlay
  • Simulated emotions: Happy, Sad, Angry, Neutral, Sleepy
  • Productivity score OOP
  • Auto study timer + log.txt logging
  • Console BEEP alerts + session summary

⚠️ This is a native C++ application. Download the repo and run it locally with your webcam.
