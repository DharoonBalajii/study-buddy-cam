Quick Start

  1. Clone the repository to your machine
  2. Install OpenCV — brew install opencv (Mac) or sudo apt install libopencv-dev (Linux)
  3. Copy haarcascade_frontalface_default.xml into the project folder
  4. Compile all source files together
  5. Run the executable — ./app

Compile & Run

# Compile all source files
g++ main.cpp face_detection.cpp emotion_simulator.cpp decision_engine.cpp productivity_tracker.cpp -o app `pkg-config --cflags --libs opencv4`

# Run the program
./app

Features

  • Real-time webcam face detection with green rectangle overlay
  • Simulated emotions: Happy, Sad, Angry, Neutral, Sleepy
  • Context-aware study suggestions based on detected emotion
  • Productivity score tracking with OOP design patterns
  • Auto study timer with session logging to log.txt
